# magma-mockups

Repository for generated mockups used by Magma.

## Purpose

This repo is a lightweight container for generated visual mockups, static previews, and shareable prototype output.

Primary uses:
- storing generated mockup artifacts
- publishing simple static previews via GitHub Pages when useful
- keeping mockup deliverables separate from product/application source repos

## Notes

- This is not intended to be the main codebase for a product.
- Treat it as an asset/output repository for mockups and presentation-ready generated material.
- Keep the structure simple and easy to browse.

## Current contents

- `index.html` — simple landing page for the GitHub Pages site
- `.nojekyll` — ensures raw static file serving on GitHub Pages

## Suggested usage

Add mockups in clearly named folders by project, client, or concept, for example:

```text
project-a/
  homepage-v1.png
  dashboard-v2.png
client-b/
  mobile-flow-01.png
  mobile-flow-02.png
```

If needed, add small index pages or asset manifests so previews are easier to navigate.
