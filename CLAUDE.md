# CLAUDE.md

## Repo purpose

This repository is used to contain generated mockups for Magma.

It should be treated as an output/assets repo for:
- generated UI mockups
- static preview files
- simple prototype presentation artifacts
- lightweight GitHub Pages hosting when convenient

## Working rules

- Prefer simple static files and folders.
- Do not turn this into a full application repo unless explicitly requested.
- Keep naming clear by client, project, or concept.
- Favor browseable structure over complex build tooling.
- When adding new mockups, preserve a clean top-level layout.

## Expected contents

Typical contents may include:
- images (`.png`, `.jpg`, `.webp`)
- static HTML previews
- small supporting CSS/JS files
- per-project folders for generated mockup sets

## Current repo state

- Minimal static landing page in `index.html`
- GitHub Pages-friendly setup via `.nojekyll`

## Documentation intent

The README should describe the repo as a generated mockups repository, not as a product codebase.
